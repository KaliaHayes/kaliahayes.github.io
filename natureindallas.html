Nature In Dallas

My Web Design & Development for Business Applications course introduced me to fundamental concepts of designing and developing responsive websites. Course topics include HTML, CSS, Domain registration, web hosting mgmt., FTP, WordPress, SEO, and Web Analytics. Throughout, I received hands-on expierence with a variety of web design and dev tools. Our major semester project required us to design and develop a dynamic website using Wordpress.

Idea

My groupmates and I decided to build a website geared towards introducing UTD students and other DFW residents to nature attractions in the metroplex. Our goal was to provide a one stop shop for finding your next Dallas adventure.

Task

As per project guidelines, we were tasked with creating two sets of wireframes early on. The first set was to be hand-drawn as an introduction to low-fi wireframing. Using the hand-drawn wireframe designs, we were required to code a hi-fi wireframe using Bootstrap. 

Key Takeaways

Web Hosting & WordPress

Because I had an idea of HTML, CSS, and Bootstrap basics upon the start of my course, my goal was to become familiar with Wordpress, web hosting, and SEO. For this project, we were instructed to use HostGator for web hosting and domain registration.

Prior to this project, my knowledge of web hosting started and ended with GitHub Pages, so this process was new to me. In fact, using what I knew about domain registration, I ended up registering my personal domain using Namecheap's free-for-students .me domain

NatureInDallas was created using the open source CMS WordPress. This was my first time using WordPress, and I am now familiar with common functionalities such as installing, navigating the dashboard, and using plugins and themes. After the quick 5 minute self-hosted installation, we used Cyberduck FTP to transfer our project files to WordPress.

In order to meet our project objective, we selected a pre-built WordPress theme and installed Boostrap CDN plugin to recreate our existing wireframes.

SEO

Once our website was cosmetically and functionally sound, attention shifted to learning and applying SEO practices. We spent a good deal of time grasping the concept of writing good content and optimizing our website for search engines. We accomplished this mannually by editting <title> and <h1> tags, generating an XML and HTML sitemap, and creating a robots.txt.

The Yoast SEO WordPress plugin allowed us to select focus keywords for our pages, define page meta descriptions, preview our website as a Google search result, and generate a sitemap. With these adjustments, the chances that a user may find our website through an organic Google search due to the sheer relevance of our website increased.
